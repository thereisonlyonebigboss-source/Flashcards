üß† What is AI Flashcard Generator?

AI Flashcard Generator is a web application that transforms your study notes into intelligent, AI-powered flashcards. Using advanced language models like Llama 2, it automatically extracts key concepts and creates question-answer pairs from your documents.

Whether you're studying biology, chemistry, history, or any subject, this tool helps you create effective study materials without the manual work of writing flashcards yourself.

üöÄ Getting Started

Setting up the AI Flashcard Generator is simple:

  1. Install Llama (choose one option):
    # Option 1: Ollama (Recommended) curl -fsSL https://ollama.ai/install.sh | sh ollama pull llama2 # Option 2: Local Llama Installation # Follow instructions at: https://github.com/facebookresearch/llama
  2. Install Python dependencies:
    pip install flask pandas openpyxl python-docx pdfplumber requests
  3. Run the web application:
    python web_app.py
  4. Open your browser:

    Navigate to http://localhost:5000 to start generating flashcards!

üîß Troubleshooting

Llama Connection Issues:

  • Ensure Llama is properly installed and running
  • Check if Ollama service is active (if using Ollama)
  • Verify firewall settings aren't blocking local connections
  • Try restarting the Llama service

File Processing Issues:

  • Ensure file formats are supported (TXT, MD, DOCX, PDF)
  • Check that files aren't password-protected
  • Verify files contain readable text content
  • Try smaller files first to test the system
